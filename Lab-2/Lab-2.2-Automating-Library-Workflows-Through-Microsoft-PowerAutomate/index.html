Lab 2.2 - Automating Library Workflows Through Microsoft PowerAutomate¶

In this lab, you'll explore how Microsoft PowerAutomate can be used to streamline everyday library tasks. By building an automated workflow that handles study room reservations, you'll learn how to connect tools like Microsoft Forms, Excel, and Outlook without writing any code. This hands-on activity shows how automation can save time, reduce errors, and improve the user experience in a library setting. Get ready to create your first real-world automation—from form submission to email confirmation!

We will be creating a workflow that detects when a study room is booked through a Microsoft Form submission, then sends a confirmation email, and adds a row to an Excel sheet!

Learning Objectives¶

After completing this project, you will be able to:

  • Understand how to create and connect Microsoft Forms, Excel, and Outlook using PowerAutomate.
  • Build an automated workflow to manage and track study room reservations in a library setting.
  • Gain hands-on experience with low-code automation tools to improve operational efficiency.

Step 4 - Use Copilot to Generate a Flow¶

On the home screen, you should see a Copilot Box where you can enter a prompt to generate a workflow! Enter the following prompt:

When a Microsoft Form is submitted to book a study room, send a confirmation email to the user and log the booking in an Excel sheet.

Then click the blue generate button!

Step 7 - Set Up Parameters¶

Now you should see a screen that looks like this:

As you can see, there are some error messages on some of the steps. This is because PowerAutomate doesn't know which Form or Excel sheet it is retrieving information from. To fix this, do the following:

  1. Click on the box that says "When a new response is submitted"
  2. In the bar on the side, click on the dropdown and select the form you created previously.
  3. Do the same for the "Get response details" box.

For the Excel step, you'll see some different dropdowns - "Location, Document Library, File, and Table."

  1. Location: The specific OneDrive where the Excel file is saved
  2. Library: OneDrive
  3. File: Navigate to the folder where the file is stored by clicking the folder icon
  4. Under Table, choose "Log" as that is the name of the table provided in the file.

Step 8 - Fix Dynamic Values!¶

The last step before testing out your flow is to fix the dynamic values. In the step where the flow is sending an email or adding a row, it retrieves data from the Microsoft Form through dynamic values (values that change every time the form is submitted). We have to fix these values to accurately reflect our form.

In the "To" Section of the Send an email step, there should be a little green box that says "body/email." However this was set before we updated our parameters, so it is outdated. Click the x next to the name to delete it.

To add the updated parameter click into the textbox and select the blue lightning bolt on the side. This opens a pop-up that allows you to select different dynamic values. However, you might not see the one you are looking for. In that case click "See More" which will reveal all the dynamic values.

Select the value that corresponds with the question where the user inputs their email into the form. Similarly, update the body of the message to reflect the changes!

To fix the Excel step, it's a little bit different. First click on the box to open up the side panel. Then select "Show All" beside "Advanced Parameters". This will open up some parameters that correspond to the columns in the table.

For each column, put the correct dynamic value into the textbox below it. You can safely ignore "DateTime Format."

Step 9 - Test it out!!¶

It's time to test your flow! First make sure to save your flow by clicking the save button on the top right corner. Then head back to the Microsoft Form you duplicated and click the "Preview" button on the top right. Fill out the form then hit submit!

Check your email to see if you get a confirmation email and check the Excel sheet to see it update (this might take a little bit, but it should be there within a minute)!
